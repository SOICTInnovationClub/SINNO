
1.	Mô tả sơ lược

Công nghệ luôn thay đổi. Để đón đầu xu thế của giới công nghệ, chúng tôi tạo ra một trang web có thể tự động cập nhật các bài báo
về khoa học và công nghệ từ nhiều trang web khác nhau. Mỗi bài viết đều được tóm tắt và trích xuất từ khóa, giúp người dùng tiếp nhận 
thông tin nhanh chóng và tiết kiệm thời gian.

2.	Kiến thức liên quan

Dự án liên quan đến : Việc trích xuất dữ liệu từ các web và mảng xử lý ngôn ngữ tự nhiên (natural language processing) trong AI.
•	Web Scraping : sử dụng framework Selenium, thư viện beautifulsoup, newspaper trong python để trích xuất dữ liệu từ các bài báo 
của nhiều trang web khác nhau
•	Natural language processing : 
-	Text summarization : dùng thư viện sumy để tóm tắt nội dung các bài báo đã trích xuất 
-	Keyword extraction : dùng thư viện KeyBERT để trích xuất từ khóa của các bài báo
•	Web programming : Dùng html và css để lập trình web
